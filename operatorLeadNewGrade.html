  6. Select Raw Bin, Percentage and choose if it need mix with other bin
  7. Add / Remove bin if needed
  8. Select Cleaning Bin, and Amount to clean or fill up to (Ring, Feet, etc.)
  9. Add / Remove bin if needed
  10. Select Color Sorter and new Delta settings
  11. Choose or Select Additional settings and leave a Comment if needed
